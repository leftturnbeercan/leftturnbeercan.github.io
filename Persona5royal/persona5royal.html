Guide to this Walkthrough for Persona 5 Royal Edition

Note: This walkthrough is for Persona 5 Royal. If playing the base game Persona 5, this guide will not be the best option to follow along with.

Romancing Options:

Starting on July 6th there will be the first romancing option decision. This guide will give you the option to romance any of the female protagonists you want. If you're interested in the Thieves Den rewards, make sure to create a save before moving on with the romance option. Max out the Confidant to get the reward. If the Confidant romancing option you max is not what you wanted, then reload from the previous save point to go back through the friendship route and continue to follow the guide.
